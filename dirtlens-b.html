Role: Product Designer

Date: 2024

Tools: Python, CAD, Open Source Computer Vision

Project Overview

DirtLENS is a compact, handheld device that helps field researchers and earth scientists capture accurate soil color and stratigraphy data in real time. It records color readings along with GPS coordinates to support efficient site documentation and sampling across diverse environments. All information is formatted for easy export and is cleared after each scan to ensure data security, speed, and readiness for the next measurement.

Problem & Research

Field archaeologists often rely on manual soil color charts and handwritten notes, which can introduce errors and inefficiencies. Interviews and field observations revealed the need for a single tool to capture, store, and export soil data securely.

Design & Prototyping

Using rapid prototyping and open-source hardware, we developed a device housing a color sensor, GPS module, and microcontroller. Multiple iterations ensured the device remained ergonomic and robust for field conditions.

Outcome & Reflection

DirtLENS shows how modern sensor design can enhance field data collection for soil profiles and stratigraphy. By combining precise color measurement with GPS tagging, it improves how researchers document and compare soil horizons across sites. Future versions will expand on secure data export and real-time cloud sharing for multi-site fieldwork.

“It’s a game-changer for field data collection — simple, secure, and reliable.” — Field Test User
